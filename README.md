## Cara Menjalankan

1. Clone repository
```bash
git clone https://github.com/username/bakso-urat-lumayan-pos.git
```

2. Install dependencies
```bash
npm install
```

3. Jalankan development server
```bash
npm run dev
```

4. Build untuk production
```bash
npm run build
```
